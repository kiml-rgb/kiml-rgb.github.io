前言

通勤时间太长真的会让人感觉想亖，大概4个多小时的往返，瘫在沙发上一动都不想动。面试了华信咨询设计研究院的开发岗，结果就是不自量力。不知道为什么所有人都在催催催，投简历，得到一个面试，然后就去了，真的很烦，其实知道自己不太行，代码写的太少很多东西都不是很会。但是没办法，所有人都在催，好像这个月找不到工作，就会亖一样，中国人有个屁的gap，每天都活得像头驴一样，被吊着往前走。大学毕业找不到工作，确实很可笑吧，又怎么样呢，确实在校期间没有好好练习，仅仅卡着及格线过线就行。确实也不知道自己在干什么。

在boss上看到的这个职位。

初级Java开发工程师
7-9K，13薪

22年计算机相关专业应届毕业生，有实习经验后端，熟练使用SSM框架

其实看了要求就知道自己不太行了，但确实应聘软件上甚至23届应届生也是这个要求，还是回到原点，太垃圾了某人。

name time
张三 2022-6-1 19:54:58
李四 2022-7-1 19:54:58
张三 2022-6-5 19:54:58

额额，没写出来，完全不知道数据库里还有DATE_FORMAT。。。

SELECT name,
    DATE_FORMAT(time,'%Y.%m') as month,
    COUNT(*) as time
FROM
    time
GROUP BY NAME, DATE_FORMAT(time,'%Y.%m')
